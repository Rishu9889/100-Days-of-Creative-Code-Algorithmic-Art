This sketch generates a geometric tessellation inspired by traditional
Islamic art patterns.

An eight-point star is constructed by overlapping two identical squares,

## Concept

Islamic tessellations rely on mathematical precision, rotational
symmetry, and infinite repetition. Complex patterns emerge from simple
geometric constructions without the use of figurative imagery.

## Techniques Used

- Geometric construction
- Rotational symmetry
- Pattern repetition (tiling)
- HTML Canvas API
