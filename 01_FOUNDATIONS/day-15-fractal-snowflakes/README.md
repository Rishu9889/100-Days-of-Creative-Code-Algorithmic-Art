This project is a creative coding experiment built using **HTML Canvas and JavaScript**.  
It generates beautiful **fractal-style snowflakes** using **recursion and symmetry**.

Each snowflake is drawn by a recursive algorithm that builds branching patterns, and the
entire shape is rotated to form a snowflake. The center snowflake gently rotates.

This visually demonstrates how complex patterns can emerge from simple rules.
