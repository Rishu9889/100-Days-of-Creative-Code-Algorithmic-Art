This sketch introduces basic physics simulation using position,

A constant gravitational force accelerates a ball downward. When the

## Concept

Motion is the result of forces acting over time. By explicitly modeling
forces, velocity, and acceleration, complex physical behavior can be
built from simple rules.

- Newtonian motion
- Gravity as a constant force
- Velocity integration
